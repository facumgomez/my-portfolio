# 💻 Facundo Gómez - Frontend Developer Portfolio

Bienvenido al repositorio de mi portfolio personal. Aquí exhibo mis proyectos, habilidades técnicas y mi formación académica. El sitio está construido con un enfoque en la experiencia de usuario (UX), el rendimiento y la escritura de código limpio y modular.

## ✨ Características Principales
- **Modo Claro/Oscuro:** Integrado globalmente utilizando `ThemeContext` de React, adaptando no solo los estilos generales sino también las alertas dinámicas.
- **Formulario de Contacto Interactivo:** Implementado con Formspree para el envío real de correos sin necesidad de backend, interceptado con Fetch API y alertas personalizadas de SweetAlert2.
- **Diseño Responsive:** Totalmente adaptable a dispositivos móviles y escritorio utilizando CSS Modules.
- **Optimización de Rendimiento:** Implementación de `lazy loading` en las imágenes de los proyectos para una carga inicial ultra rápida.
- **Accesibilidad (a11y):** Estructura semántica, uso de botones para elementos interactivos y etiquetas `sr-only` para lectores de pantalla.

## 🚀 Tecnologías Utilizadas
- **React (Vite):** Para una construcción rápida y eficiente de la interfaz.
- **CSS Modules:** Estilos encapsulados, evitando colisiones de clases y facilitando el mantenimiento.
- **SweetAlert2:** Para el manejo de notificaciones de estado y modales.
- **Formspree:** Como servicio de integración para la recepción de correos electrónicos.

## 🛠️ Instalación Local

Si deseas probar o inspeccionar este proyecto en tu entorno local, sigue estos pasos:

1. Clona este repositorio:
   ```bash
   git clone [https://github.com/facumgomez/my-portfolio.git](https://github.com/facumgomez/my-portfolio.git)
   ```

2. Ingresa al directorio del proyecto:
   ```bash
   cd my-portfolio
   ```

3. Instala las dependencias necesarias:
   ```bash
   npm install
   ```

4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
